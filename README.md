## Sobre o App

Breve explicação do que o app faz.  
Criamos um app do nosso Studio de tatuagem e aplicação de piercing, para os clientes verem as opções de joias disponiveis e modelos de tatuagem e piercings, também agendamos um horario para o seu atendimento onde você pode enviar ideias do seu gosto. Mostramos os trabalhos dos nossos profissionais.
